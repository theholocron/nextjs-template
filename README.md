# NextJS Template

A NextJS template used within the Galaxy.

## Installation

```bash
npm install --save-dev @theholocron/nextjs-template
```

## Usage

```typescript
import { Component, type ComponentProps } from "@theholocron/nextjs-template";

function App (props: ComponentProps) {
	return (
		<Component {...props} />
	);
}
```
